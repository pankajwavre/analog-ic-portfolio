## 📌 About This Repository

This repository presents a curated selection of **transistor-level analog and mixed-signal IC design projects** completed as part of graduate-level coursework in:

- Analog Integrated Circuit Design (AIC)
- Advanced Analog IC Design (AAIC)
- Switched-Capacitor & Sigma-Delta Systems

All projects are designed and verified using **Cadence Virtuoso / Spectre**, with emphasis on:
- Circuit architecture
- Biasing and stability
- Large- and small-signal performance
- Realistic CMOS implementation

📁 `folded-cascode-ota/`
